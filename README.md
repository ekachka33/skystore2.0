# Интернет-магазин на Django

Проект интернет-магазина, который разрабатывается в рамках курса. В процессе курса будет создано приложение для каталога товаров, настроены шаблоны для страниц, реализованы контроллеры для обработки запросов, а также добавлена форма обратной связи на страницу контактов. Проект поможет освоить основные принципы разработки веб-приложений на Django и применить полученные знания на практике.

## Задачи проекта:
1. Создание и настройка проекта Django.
2. Разработка приложения для каталога товаров.
3. Реализация HTML-шаблонов для страниц.
4. Настройка маршрутизации и создание контроллеров для отображения страниц.
5. Реализация формы обратной связи на странице контактов (дополнительная задача).
